    • Nyxmoore Reach

    A game I am currently working on is called Nyxmoore Reach. It's going to be a sandbox farming game with a story line behind it. It is highly inspired by multiple games. One of those examples being Stardew Valley. This was also the first ever project where I made an engine script. The gist of the game will be that you, the player, moves to a new city that has long stopped believing in witchcraft. You will have to climb your way up and get a good reputation to attract more customers. Make sure to befriend the people in the city itsself.

    Programs: Unity 6, Procreate, Aseprite

    Inventory Scriptable Object

    My initial idea was to make multiple bases for the different types of items, but I realised: "Hey.. Wouldn't it be easier if I just created a dropdown menu with item type?" And so I started on that. I soon ran into the issue that certain items did not need a grow season. For example, furniture. Because yes, let me just grow a lamp tree. So that's where I created my first ever engine script.

    There's four different item types: herbs, fruits, animal drops and furniture. I want all of these to have different abilities, so the different properties will come in extremely handy in the near future. To explain the engine script, the image will be listed below with some more context, we get the enum names, And set it as a variable with the type "SerializedProperty". We draw the properties, with that I mean that I load them in, EXCEPT the ones that have been saved in the variable. We go through a few if statements, and we check which properties we want to show with each type. (e.g. If the type is herb, show these properties, and that for every type.)
